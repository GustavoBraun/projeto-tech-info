### Requirements

- Node.js v10+

#### Environment Variables

- PORT: 8080
### Running

- Instale as dependências  - `npm i`
- Execute o projeto - `npm run dev`
- Execute os testes - `npm teste`
- Vá para a página do swagger - ` http://localhost:8080/api-docs/`
